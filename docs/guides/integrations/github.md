# GitHub

Analyze repositories and bootstrap your [feature catalog](/product).

---

## Connect

1. **Knowledge > Integrations** → **GitHub** → **Connect GitHub**
2. Install the GitHub App and grant repo access
3. On the manage page, enable **Analyze** on repos you want indexed

| Action | Purpose |
|--------|---------|
| **Analyze** | Include repo in catalog and analysis jobs |
| **Rescan** | Re-run analysis after large changes |
| **View analysis** | Tech stack and structure Devplan extracted |
| **Refresh** | Sync repo list after GitHub App changes |

First analysis often takes **30+ minutes**.
