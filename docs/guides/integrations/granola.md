# Granola

Sync Granola call notes and meeting summaries into your workspace.

---

## Requirements

Granola API keys are only available on **paid Granola accounts**. Free accounts cannot connect due to Granola API limitations.

---

## Setup

1. Open **Knowledge > Integrations** → **Granola**.
2. Enter your **Granola API key** (workspace **Admins** only).
3. Add **folder names** to match folders in Granola.
4. Save.

Shows **Connected** when at least one folder is configured.

---

## What it feeds

Synced notes can appear in [Signals](/signals), [Insights](/insights), and [Today](/today).
