# Bitbucket

Connect Bitbucket to analyze repositories and bootstrap your [feature catalog](/product).

---

## Prerequisites

Install the [Devplan Forge app for Bitbucket](https://developer.atlassian.com/console/install/7fe2abea-86cb-4bec-a331-b81d65fb6c62) on your Bitbucket workspace before connecting. Forge apps require a **shared team workspace** (not a personal workspace).

---

## Connect

1. Open **Knowledge > Integrations** → **Bitbucket** → **Connect**.
2. Complete Bitbucket OAuth.
3. On the manage page, enable **Analyze** on repos you want indexed.

| Action | Purpose |
|--------|---------|
| **Analyze** | Include repo in analysis and catalog jobs |
| **Rescan** | Re-run analysis after large changes |
| **View analysis** | See extracted structure and tech stack |
| **Refresh** | Sync repo list after permission changes |

First analysis often takes **30+ minutes**.
