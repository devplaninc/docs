# Settings Overview

Open **Settings** from your profile menu (bottom-left). Use **Back to app** to return to your workspace.

---

## Profile

| Page | What it does |
|------|--------------|
| [Personal Info](/personal-info) | Name, email, workspace memberships |
| [Preferences](/preferences) | Roles, appearance, IDE, digest email |

---

## Workspace

| Page | What it does |
|------|--------------|
| [General](/workspace-general) | Name, logo, website, deletion |
| [Members](/workspace-members) | Invites and roles (Admin / User) |
| [Daily Digest](/workspace-daily-digest) | Workspace digest instructions and Slack delivery |
| [API Keys](/workspace-api-keys) | API keys for MCP access |

---

## Organization

Visible when you belong to an organization.

| Page | What it does |
|------|--------------|
| [General](/organization-general) | Org name, logo, owners |
| [Workspaces](/organization-workspaces) | Manage org workspaces |
| [Billing](/organization-billing) | Subscription |
