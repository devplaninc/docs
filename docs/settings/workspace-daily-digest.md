# Workspace Daily Digest

Workspace-wide digest settings for [Today](/today). **Settings > Workspace > Daily Digest**

- **Default instructions** — how the workspace digest is generated
- **Generate** — manual 24-hour report (in-app only, not emailed)
- **Delivery time** — email schedule (local timezone)
- **Email recipients** — all members or role defaults
- **Slack channel** — requires Slack connected under [Integrations](/integrations)

Per-user email opt-out: [Preferences](/preferences).
