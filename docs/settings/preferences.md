# Preferences

**Settings > Profile > Preferences**

- **Roles** — personalize [Insights](/insights); saves automatically
- **Preferred IDE** — for CLI flags when specs are enabled
- **Daily Digest email** — toggle per workspace (same setting as **Manage communication settings** on [Today](/today))
- **Color theme** — Light, Dark, or System
