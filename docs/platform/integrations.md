# Integrations

Connect external data sources that feed knowledge, signals, and project context across Devplan. **Integrations** is the hub for linking your code repos, docs, tickets, and communication tools.

---

## How to access

Click **Integrations** under the **Knowledge** section in the left sidebar.

---

## Key features

- **Provider cards** — each integration shows its name, description, and connection status
- **Status badges** — **Connected**, **Configured**, or **Available**
- **Actions** — **Connect**, **Configure**, or **Select folders** depending on the provider and state
- **Search** — filter the integration list by name

## Common workflows

1. Open **Integrations** and review which sources are already connected.
2. Click **Connect**, **Configure**, or **Select folders** on the provider you need.
3. Complete provider-specific setup — enable repos, attach pages, select folders, invite the Slack bot, or upload files. Connecting alone is often not enough; see the guide for each provider.
4. Return to **Signals** or **Insights** after connecting new sources to see new data flow in.
